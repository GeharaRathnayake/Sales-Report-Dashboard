# Sales-Report-Dashboard
A comprehensive Power BI dashboard analyzing global sales performance using real business data. Includes geographic analysis, product profitability, segment breakdown, and year‑over‑year trends. Created using Power BI, Power Query, and DAX with fully interactive visuals and business insights.
## Project Overview
This project provides a comprehensive analysis of:
- Total Sales
- Gross Margin
- Margin %
- Segment Performance
- Product Profitability
- Country-wise Sales
- Year-over-Year Revenue Trends

## Dataset Description
The dataset contains two sales-related tables with fields such as:
- Country
- Product
- Segment (Government, Enterprise, Small Business, etc.)
- Year, Month, Quarter
- Total Sales
- Gross Margin
- Margin %
- Units Sold
- Discounts & Pricing Fields

## Business Questions Answered
1. Which countries generate the highest total sales and gross margin?
2. What is the year-over-year trend in Total Sales and Gross Margin?
3. Which customer segments contribute most to revenue?
4. Which products deliver the highest profitability?
5. How does Margin% vary across countries and product lines?
6. Are there seasonal or monthly sales patterns?
7. What insights can leadership use to improve market strategy?

## Tools & Technologies Used
- Power BI Desktop
- Power Query (ETL & data cleaning)
- DAX (Calculated measures)
- Excel (Source Data)
- Data Modeling Techniques
- Interactive Visualizations

## Data Preparation Process
1. Imported the dataset into Power BI
2. Cleaned and transformed data using Power Query
3. Ensured correct data types and removed inconsistencies
4. Built a Date Table and established relationships
5. Created DAX measures for Total Sales, Gross Margin, Margin %
6. Standardized data formats & designed visuals
7. Added slicers for interactive filtering (Year, Segment)

## Dashboard Visualizations
- Map: Total Sales by Country
- Area Line Chart: Total Sales, Gross Margin & Margin% over time
- Donut Chart: Gross Margin by Product
- Pie Chart: Gross Margin by Year
- Bar Chart: Total Sales by Segment
- Column Chart: Sales & Margin% by Country
- Interactive Filters: Segment, Product, Year

## Key Insights
- Government segment produced the highest sales
- 2014 outperformed 2013 in total revenue
- United States, Germany, and France were highest-earning regions
- Products like Paseo, Velo, and Amarilla contributed heavily to gross margin
- Some products had strong sales but lower profitability
- Clear seasonal patterns observed in mid-2014
- Europe & North America regions consistently led performance

## Conclusion
This Power BI dashboard provides a clear and interactive overview of global sales performance. It enables organizations to optimize strategy, identify strong/weak markets, improve product offerings, and support financial planning.
